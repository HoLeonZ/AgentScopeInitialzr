## 🚀 快速部署（三步完成）

### 第一步：上传文件到服务器

```bash
# 方式1：使用 scp 上传整个目录
scp -r agentscope-initializr-arm64 user@your-server:/tmp/

# 方式2：先上传 zip 包，然后在服务器解压
scp agentscope-initializr-arm64.zip user@your-server:/tmp/
```

### 第二步：SSH 登录到服务器并解压

```bash
# 登录服务器
ssh user@your-server

# 进入目录
cd /tmp

# 如果是 zip 包，先解压
unzip agentscope-initializr-arm64.zip
cd agentscope-initializr-arm64
```

### 第三步：运行部署脚本

```bash
# 添加执行权限
chmod +x *.sh

# 运行部署脚本
./deploy-arm64.sh
```

**就这么简单！** 服务会自动启动并运行在 http://your-server-ip:8000

## 🔧 系统要求

- **架构**: ARM64 (aarch64)
- **操作系统**: Linux (Ubuntu 20.04+, Debian 10+, CentOS 7+)
- **内存**: 最低 512MB，推荐 2GB+
- **磁盘**: 最低 1GB 可用空间

## 📝 常见问题

### Q1: 脚本没有执行权限？
```bash
chmod +x *.sh
```

### Q2: Docker 或 Docker Compose 未安装？
使用 `quick-start-arm64.sh` 脚本会自动安装，或手动安装：
```bash
# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 安装 Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose
```

### Q3: 端口 8000 被占用？
编辑生成的 `docker-compose.arm64.yml`，修改端口映射：
```yaml
ports:
  - "8080:8000"  # 改为其他端口
```

### Q4: 如何设置开机自启？
```bash
# 方式1：使用系统服务
sudo cp agentscope.service /etc/systemd/system/
sudo systemctl enable agentscope.service
sudo systemctl start agentscope.service

# 方式2：使用 Docker Compose 的 restart 策略
# 已在 docker-compose.arm64.yml 中配置：restart: unless-stopped
```

### Q5: 如何查看服务日志？
```bash
# 实时日志
./manage.sh logs

# 或者使用 Docker 命令
docker-compose -f docker-compose.arm64.yml logs -f
```

### Q6: 服务启动失败怎么办？
```bash
# 1. 查看详细日志
./manage.sh logs

# 2. 检查容器状态
docker ps -a

# 3. 运行健康检查
./health-check.sh

# 4. 重新部署
./deploy-arm64.sh
```

## 🔐 生产环境建议

### 1. 配置防火墙
```bash
# Ubuntu/Debian
sudo ufw allow 8000/tcp
sudo ufw enable

# CentOS/RHEL
sudo firewall-cmd --permanent --add-port=8000/tcp
sudo firewall-cmd --reload
```

### 2. 使用 Nginx 反向代理
```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### 3. 配置 SSL/HTTPS
使用 Let's Encrypt 免费证书：
```bash
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```

### 4. 定期备份
```bash
# 备份输出目录
tar -czf backup-$(date +%Y%m%d).tar.gz ./output/

# 设置定时任务
crontab -e
# 添加：每天凌晨 2 点备份
0 2 * * * tar -czf /backup/agentscope-$(date +\%Y\%m\%d).tar.gz /path/to/output/
```
