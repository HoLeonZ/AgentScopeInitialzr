## 交互流程

### 配置流程
```
用户操作流程：

1. 点击"知识库"（左侧第4项）
   ↓
   右上切换为"📚 Knowledge Base Configuration"
   ↓
   填写配置：
   • 选择知识库类型
   • 配置向量数据库
   • 上传文档
   • 设置检索参数
   ↓
   点击"下一步"或直接切换到其他配置
   ↓
   知识库状态变为 ✓
```

### 状态更新
```
未配置时：
▶ 知识库 ⏳

配置中：
▶ 知识库 ⏳ (部分配置)

完成配置：
▶ 知识库 ✓
```

## 关键改进

### 1. 更清晰的命名
- ✅ RAG → 知识库（更直观）
- ✅ Skills → Skill 配置（简洁）
- ✅ 移除百分比，状态更清晰

### 2. 更合理的顺序
```
基础 → 模型 → 记忆 → 知识库 → Skills → 扩展 → 测试
  必填   必填   推荐    可选    可选   可选    推荐
```

### 3. 简化的状态显示
- 去掉所有百分比数字
- 只显示 ✓ 或 ⏳ 状态
- 界面更加清爽

### 4. 专业的分类
- 知识库：专注于数据和检索
- Skill：专注于能力扩展
- 扩展：专注于工具和格式化

这个设计让配置流程更加清晰专业！
