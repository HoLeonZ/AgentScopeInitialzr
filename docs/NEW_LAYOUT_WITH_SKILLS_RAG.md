## 菜单项优先级和顺序

### 推荐配置顺序
```
1. 📋 项目基础          → 必填
   ↓
2. 🤖 模型配置          → 必填
   ↓
3. 💾 记忆配置          → 推荐
   ↓
4. 🛠️ Skills            → 可选
   ↓
5. 📚 RAG               → 可选
   ↓
6. 🔧 扩展功能          → 可选
   ↓
7. 📊 测试评估          → 推荐
```

### 完成度计算
```
总体完成度 = (已完成项总数 / 总项数) × 100%

示例：
- 完成 2 项 = 2/7 = 29%
- 完成 4 项 = 4/7 = 57%
- 完成 7 项 = 7/7 = 100%
```

## 交互说明

### 展开/收缩逻辑
```
默认状态：
- 所有配置项收缩
- 显示完成度指示

点击展开：
- 显示子项（如果有）
- 自动滚动到对应配置区

收缩其他：
- 点击其他配置项时自动收缩当前项
- 保持界面简洁
```

### 完成度显示
```
未配置：⏳ (0%)
部分配置：⏳ (50%)
已完成：✓ (100%)
```

这个设计让配置流程更加清晰和专业！
